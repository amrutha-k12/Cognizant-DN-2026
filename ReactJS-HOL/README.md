# React Components

## React Components
- Reusable and independent UI building blocks in React.

## Difference Between Components and JavaScript Functions
- Components return JSX to display UI, while JavaScript functions perform general tasks and return values.

## Types of Components
- Class Components
- Function Components

## Class Component
- An ES6 class that extends `React.Component` and uses the `render()` method.

## Function Component
- A JavaScript function that returns JSX.

## Component Constructor
- A special method used to initialize state and bind methods in a class component.

## render() Function
- A method that returns the JSX to be displayed on the screen.
