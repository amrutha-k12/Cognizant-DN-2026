# Component Lifecycle in React

## Need and Benefits of Component Lifecycle

The Component Lifecycle allows developers to execute code at different stages of a component's existence, such as creation, updating, and removal. It helps in data fetching, performance optimization, and resource management.

## Lifecycle Hook Methods

Some commonly used lifecycle methods are:

- `constructor()`
- `render()`
- `componentDidMount()`
- `componentDidUpdate()`
- `componentWillUnmount()`

## Sequence of Rendering a Component

1. Constructor is called.
2. Render method executes.
3. Component is mounted to the DOM.
4. `componentDidMount()` is executed.
5. Component updates when state or props change.
6. `componentDidUpdate()` is executed.
7. `componentWillUnmount()` executes before component removal.
