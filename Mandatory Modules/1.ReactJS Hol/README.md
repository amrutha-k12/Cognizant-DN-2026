# React Introduction

## Define SPA and its Benefits
A **Single Page Application (SPA)** loads a single HTML page and updates content dynamically without reloading the entire page, providing faster performance and a better user experience.

## Define React and Identify its Working
**React** is a JavaScript library for building user interfaces using reusable components and updating the UI efficiently through the Virtual DOM.

## Identify the Differences Between SPA and MPA
**SPA** loads one page and updates content dynamically, whereas **MPA (Multi-Page Application)** loads a new page from the server for each request.

## Explain Pros & Cons of Single-Page Application
SPAs provide faster navigation and a smoother user experience but may have slower initial loading and SEO challenges.

## Explain About React
React is an open-source JavaScript library developed by Meta for building interactive and component-based user interfaces.

## Define Virtual DOM
The **Virtual DOM** is a lightweight copy of the real DOM that React uses to optimize and speed up UI updates.

## Explain Features of React
React features reusable components, Virtual DOM, JSX, one-way data flow, and efficient rendering for building scalable web applications.
